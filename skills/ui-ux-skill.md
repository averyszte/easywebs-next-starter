# UI/UX Skill

## Design philosophy

- Premium but approachable; direction depends on the client's brand and the app's users;
  conversion-focused; mobile-first; clean spacing; strong hierarchy. **Custom, not the default shadcn look.**

## Layout

- Avoid clutter; use whitespace intentionally; keep sections/panels visually distinct; readable typography.

## Navigation

- Sticky headers where appropriate; clean mobile nav; CTAs visible but not intrusive. For the app, a
  clear dashboard **sidebar** with obvious active state.

## Components

- Consistent card + button hierarchy; smooth hover/focus states; responsive grids. shadcn (Radix)
  gives you accessibility for free — don't break it. Theme components to the client's tokens.

## Avoid

- Generic SaaS sections, excessive gradients/animations, tiny text, crowded headers, default shadcn styling.
